# Variables

## Declaring a Variable

To declare a variable, we use the `var` keyword and a valid identifier (name):
```javascript
    var num;
    //or
    var full_name = "cory";
```

## Reassigning a Variable

To reassign a variable, we drop the `var` keyword and put in a new value:
```javascript
    var num = 5;
    num = 7
    num = num + 7 // 14
```
